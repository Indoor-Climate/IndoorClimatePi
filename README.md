# IndoorClimatePi

## Setup

Clone repo to Raspberry Pi and open folder
```bash
git clone https://github.com/MikkelFilip/IndoorClimatePi.git && cd IndoorClimatePi/
```
Change config
```bash
nano config
```
Setup the Raspberry Pi
```bash
sudo make
```
\*If the system needs to be taken down use:\*
```bash
sudo make clean
```
