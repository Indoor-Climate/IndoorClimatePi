# IndoorClimatePi

## Setup

Set parameters
```
In ./config
&  ./code/env.py
```

Clone repo to Raspberry Pi and open folder
```bash
git clone https://github.com/Indoor-Climate/IndoorClimatePi.git && cd IndoorClimatePi/
```
Change config
```bash
nano config
```
Setup the Raspberry Pi
```bash
sudo make
```
\*If the system needs to be taken down use:\*
```bash
sudo make clean
```
